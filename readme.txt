== Installation ==

This section describes how to install the plugin and get it working.

1. Make sure you have installed [Alex Rabe's](http://alexrabe.de/) - [NextGEN Gallery Plugin](http://wordpress.org/extend/plugins/nextgen-gallery/)
1. Upload folder `nextgen-resize` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. There would an option available in the Gallery tab, called 'Resize on Uploads', from where you can fine tune settings.
